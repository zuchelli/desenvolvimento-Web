    Tag 1: Del

    (Deleted Text) - Este elemento é usado para representar texto que foi removido
    ou deletado de um documento. Ele pode ser útil em situações como edições em documentos
    colaborativos ou históricos de revisões em documentos.

    Exemplos:

    DELETADO

    O arquivo 1TRIM esta removido

    Deletar arquivo ZUCHELLI...
    -Arquivo ZUCHELLI deletado

    Tag 2: Ins

    (Inserted Text) - Ao contrário do del, o ins é usado para representar texto que foi inserido em um documento.
    Isso pode ser útil para mostrar adições ou alterações em documentos, especialmente em contextos colaborativos.

    Exemplos:

    ARQUIVO ZUCHELLI ALTERADO

    O arquivo 1TRIM esta modificado

    Alterar arquivo ZUCHELLI...
    -Arquivo ZUCHELLI alterado

    Tag 3: Sub

    (Subscript) - O elemento sub é usado para definir texto que deve ser exibido como subscrito, ou seja,
    posicionado ligeiramente abaixo da linha de texto normal. É comumente usado para representar fórmulas
    químicas, equações matemáticas e notas de rodapé.

    Exemplos:

    Para fórmulas usamos sub assim Fórmula

    A fórmuloa da água é H2o

    Alterar fórmula H2o...
    -Fórmula H2o alterada

    Tag 4: Sup

    (Superscript) - Ao contrário do sub, o sup é usado para definir texto que deve ser exibido como sobrescrito,
    ou seja, posicionado ligeiramente acima da linha de texto normal. Isso é comumente usado para expoentes
    em fórmulas matemáticas, datas e notas de rodapé.

    Exemplos:

    ZUCHELLI4

    ZUCHELLI QUER ESTUDARZUCHELLI quer ir para a casa

    Eleve o número 44 ao quadrado...
    Número 442 elevado ao quadrado

    Tag 5: Blockquote

    (Block Quotation) - Este elemento é usado para definir um bloco de texto que é uma citação longa ou destacada.
    Ele é geralmente usado para citações de outros autores ou fontes.

    Exemplos:

    Autor

    ZUCHELLI, João G.

    Este texto esta em blockquote

    Tranformar a palavra Artigo em blockquote... Pronto:

    Artigo

    Tag 6: Q

    (Inline Quotation) - O elemento q é usado para definir uma citação curta dentro do texto. Ele é ideal para
    pequenas citações que devem ser incorporadas ao texto principal.

    Exemplos:

    FIRST

    A palavra mouse vem do Inglês

    Como Soócrates cita... Só sei que nada sei

    Tag 7: Abbr

    (Abbreviation) - Este elemento é usado para definir uma abreviação ou acrônimo. Ele fornece uma maneira
    de fornecer uma expansão ou explicação do significado de uma abreviação quando o cursor é passado sobre ela.

    Exemplos:

    É REAL a moeda de troca

    Estava caminhando...

    S.O.S.

    Tag 8: Address

    (Contact Information) - O elemento address é usado para definir informações de contato para o autor ou
    proprietário de um documento ou artigo. Isso pode incluir um endereço de e-mail, endereço físico,
    número de telefone, etc.

    Exemplos:

    Rua Aquela la. Bairro esse daí. Número 11

    O telefone do Professor é

    4299966611

    O e-mail de Enzo é

    joao.zuchelli.ifpr@gmail.com

    Tag 9: Cite

    (Citation) - O elemento cite é usado para definir o título de uma obra citada, como um livro, filme,
    artigo ou obra de arte.

    Exemplos:

    Noite estrelada

    Artigo sobre Van Gogh

    Transforme o título Memórias póstumas de Brás Cubas em arquivo cite
    Memórias póstumas de Brás Cubas

    Tag 10: Bdo

    (Bi-Directional Override) - Este elemento é usado para definir a direção do texto, substituindo
    a direção padrão do texto em HTML. É útil em casos onde o texto deve ser exibido em uma
    direção diferente da habitual, como em textos bidirecionais.

    Exemplos:

    Enzo indo da direita para a esquerda

    Enzo indo da esquerda para a direita

    A palavra avon ao contrario fica nova

    Tag 11: Code

    (Code) - O elemento code é usado para definir um trecho de código, como HTML, CSS, JavaScript,
    ou qualquer outra linguagem de programação.

    Exemplos:

    pink=11

    O nome enzo é uma variavel

    JavaScript

    Tag 12: Pre

    (Preformatted Text) - O elemento pre é usado para definir um bloco de texto pré-formatado, onde o espaçamento
    e quebras de linha são preservados exatamente como estão no código-fonte.

    Exemplos:

    ZUCHELLI

    Atualizar 

    Java

    Transforme o arquivo 1TRIM em arquivo pre
    Arquivo 

    1TRIM
    transformado

    Tag 13: Kbd

    (Keyboard Input) - Este elemento é usado para definir um trecho de texto que representa entrada do teclado,
    como teclas pressionadas ou atalhos de teclado.

    Exemplos:

    Tecla F1

    Segure Alt e aperte F4

    Aperte a tecla Enter...
