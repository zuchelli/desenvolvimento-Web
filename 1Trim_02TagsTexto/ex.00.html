    Tecnologias utilizadas

    Serão utilizadas as tecnologias a seguir :
    Java Script para programação e animação de páginas; CSS estiliza dando cor, sombras e estilo; HTML onde faz as marcações de hiper links colocando imagens, videos e textos das páginas; PHP uma linguagem usada para desenvolver e que tem suporte em diversos banco de dados;
    PHYTON uma linguagem de programação usada;MYSQL usado para conecção com banco de dados;

    Front-End

    Faz as interfaces que o usuario verá estilizando a própria e dando interacões possiveis ao usuario usando HTML, CSS, JAVASCRIPT.

    Back-End

    Faz os codigos para conectar a interface ao banco de dados dando segurança aos dados e as páginas usando JAVA, PHYTON, PHP.

    FullStack

    Faz a as interfaces, a modelagem e a conecção das interfaces com o sistema usando todas as tecnologias do desenvolvedor back end e front end.
