Caixas de Dialogo em Js

Sao tres tipos de caixa de dialogos.

  • alert: exibe uma mensagem de alerta para o usuario
  • confirm: exibe uma mensagem de confirmacao ao usuario em que ele pode escolher confirmar ou nao ok ou cancelar
  • prompt: o usuario pode entrar dados na caixa de dialogo

É importante destacar que o Java script pode ser incorporado ao documento Html pela tag script. O script JS pode ser interno, ou seja,dentro do documento ou externo, no qual será um arquivo externo js. Ainda é possivel colocar comandos js associados a eventos como no ex a seguir.
