    • Hyperlinks

    São referencias dentro de um documento em hypertexto para outras partes do mesmo documento ou outros documentos .

    Os links podem ser :

    • Remotos
    • Internos
    • Locais
    • Endereço de correio eletronico
    • Arquivos para download

    Hyperlinks remotos

    São hyperlinks que ligam para outros documentos disponiveis na web. Exemplo: Clique aqui!

    Hyperlinks locais

    São hyperlinks hospedados em um mesmo servidor. Exemplo: Hyperlink local

    Hyperlinks imagens

    é possivel deixar uma imagem clicavel, direcionando para uma outra referencia.
