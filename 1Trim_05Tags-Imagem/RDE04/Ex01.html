    • HTML

    É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee.

    A HTML é utilizada para publicação de conteúdo em hipertexto, tais como: [lista][negrito] texto,imagem,áudio,vídeo.
    Estes elementos podem ser [italico]vinculados através de [enfase]hyperlinks.

    A invenção do www foi fundamental ([enfase]World Wide, Web), pois trata-se de um sistema de documentos em hipermídia que são interligados e executados na Internet. [fonte pequena] fonte: Wikipedia

    Tim Berners-Lee teve grande contribuição para a evolução da internet, através da criação das seguintes tecnologias:

    • WWW
    • HTML
    • HTTP

    TAGS

    As tags definem ao navegador a marcação de conteúdo.
    As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

    As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.
